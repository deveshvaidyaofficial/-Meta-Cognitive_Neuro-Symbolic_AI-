# Adaptive Meta-Cognitive Neuro-Symbolic AI for Urban Traffic Flow Prediction in Edge-

Enabled Transportation Systems The project is aimed at building a Neuro-Symbolic AI hybrid approach to predicting urban traffic flows and traffic congestions in intelligent transportation networks. In addition to using a deep learning-based prediction model, the project also implements a Meta-Cognitive Evaluation Module that will evaluate the following prediction confidence,sensor data quality,traffic rules compliance and context validity (weather, accidents, events).Thus, this solution makes it possible to analyze the validity of the predictions made and improve traffic forecasting in intelligent transportation management.
## Data sets: 

#### 1. METR-LA Dataset – Data on traffic speeds recorded by 207 sensors in Los Angeles. 

#### 2. PEMS-BAY Dataset – Data set on traffic sensors in the San Francisco Bay Area. 

#### 3. PeMS (Performance Measurement System) Dataset – Real-world data on traffic flow, speed, and occupancy collected from California highways.
